# ParentSquare Feature Spec — School-to-Home Communication

## Overview
ParentSquare is a unified K-12 communication platform designed for schools and districts to streamline communication between administrators, teachers, and families. It provides a single interface for all school-to-home interactions, from emergency alerts to permission slips.

## Core Features

### 1. Multi-Channel Messaging
*   **Mass Notifications:** Send announcements via email, text (SMS), app notifications, and voice calls.
*   **Urgent & Smart Alerts:** Emergency notifications that bypass user "digest" settings for instant delivery.
*   **Two-Way Messaging:** Teachers and staff can communicate directly with individuals or groups without sharing personal phone numbers.
*   **StudentSquare:** A safe, secure environment for communicating with middle and high school students.

### 2. Family Engagement Tools
*   **Volunteering & Sign-Ups:** Online forms for volunteers, supply requests, and potluck sign-ups.
*   **Permission Slips:** Digitally sign and track permission slips within the platform.
*   **Forms & Polls:** Custom forms for surveys, data collection, and feedback.
*   **Conference Scheduling:** Automated parent-teacher conference sign-ups with reminders.

### 3. Translation & Equity
*   **Automatic Two-Way Translation:** Real-time translation for messages, posts, and forms in over 100 languages.
*   **Language Preferences:** Parents can set their preferred language for all communications.
*   **Website Translation:** Integrated website translation (via Weglot partnership) to keep multilingual content current.

### 4. Administrative Features
*   **District-Wide Oversight:** Centralized dashboard to monitor engagement and manage communication across multiple schools.
*   **Secure Document Delivery:** Safely send report cards, progress reports, and other sensitive documents.
*   **Virtual Phone:** Allows staff to make and receive calls from a school-provided number.
*   **Attendance & Lunch Alerts:** Automated notices for student absences and low lunch balances.

## Integrations
*   **SIS:** PowerSchool, Aeries, Infinite Campus, Skyward, and Blackbaud.
*   **LMS:** Google Classroom, Canvas, and Blackboard.
*   **General:** Google Calendar, Microsoft Office 365, and social media (Facebook/Twitter).

## Pricing
*   **Model:** Tiered packages (Engage, Engage Premium) based on student enrollment.
*   **Fees:** One-time onboarding fee + annual per-student subscription.
*   **Positioning:** Focuses on cost savings by consolidating multiple tools (e.g., Remind, Constant Contact, survey tools) into one platform.

## Pain Points
*   **Consolidation Fatigue:** Moving an entire district to one tool can be a massive cultural shift for teachers used to their own apps.
*   **App Dependency:** Full features (like forms/permission slips) are most effective when families use the app, which can be a barrier for some populations.
*   **Cost:** While it replaces multiple tools, the total annual cost can still be a significant budget line for smaller schools.

## Competitive Intelligence

### ParentSquare vs. Remind
| Feature | ParentSquare | Remind (Chat/Hub) |
| :--- | :--- | :--- |
| **Primary Focus** | Whole-School Engagement | Teacher-Parent Messaging |
| **Docs/Slips** | Built-in Permission Slips | Limited (File Sharing) |
| **Translation** | Robust (100+ languages) | Deep (90+ languages) |
| **Admin Oversight** | Very High (District Level) | High (Hub/Premium Level) |
| **Engagement** | Sign-ups/Volunteers/Forms | Mostly Messaging-Centric |

*   **Key Advantage:** ParentSquare is the platform for districts that want to eliminate "app clutter" and manage everything (from emergency alerts to lunch balances) in one secure, district-controlled ecosystem.
