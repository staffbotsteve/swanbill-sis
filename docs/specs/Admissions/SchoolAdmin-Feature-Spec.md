# Feature Spec: SchoolAdmin (Finalsite Enrollment)

## 1. Overview
SchoolAdmin is a premium admissions CRM designed specifically for K-12 private and independent schools. Following its 2021 acquisition, it is now the core of "Finalsite Enrollment," focusing on the student journey from inquiry to re-enrollment.

## 2. Feature List
### Admissions CRM
- **Inquiry Management:** Dynamic, mobile-friendly forms to capture prospective student data.
- **Lead Nurturing:** Automated, personalized email campaigns to stay in front of families.
- **Event Scheduling:** Integrated tools for scheduling school tours, "shadow days," and interviews.
- **Prospect Management:** Comprehensive dashboard to track the entire funnel (Inquiry → Applicant → Enrolled).

### Application Workflows
- **Online Applications:** Paperless, customizable forms that save parents' progress.
- **Application Tracking:** Parent-facing checklists to see missing documents (transcripts, recommendations).
- **Online File Reading:** Digital review and scoring for admissions committees.
- **Fee Management:** Automated application fee collection and waiver codes.

### Enrollment & Re-Enrollment
- **Contract Management:** Dynamic enrollment contracts with legal e-signatures.
- **Tuition Billing:** Integrated billing and payment plans for newly enrolled families.
- **Continuous Enrollment:** Support for "evergreen" contracts to simplify the annual re-enrollment cycle.
- **Parent Portal:** A single secure login for families to manage all admissions and enrollment tasks.

## 3. Pricing
- **Custom Quoting:** Finalsite Enrollment is an enterprise-grade product; pricing is based on school size, student volume, and feature modules.
- **Bundled Pricing:** Often discounted when bundled with Finalsite's CMS or marketing services.

## 4. Integration Patterns
- **Finalsite CMS:** Deep integration for a unified website and enrollment experience.
- **SIS Partners:** Native "two-way" sync with Blackbaud, PowerSchool, and Veracross.
- **API Access:** Robust REST API for custom data flows into other school systems.
- **GG4L Integration:** Connects to Infinite Campus and other OneRoster-compliant systems.

## 5. Pain Points
- **Post-Acquisition Integration:** Some schools report "legacy" SchoolAdmin features are still being merged with the Finalsite platform.
- **High Cost:** Typically too expensive for smaller K-12 schools; geared toward private/independent institutions.
- **Implementation Time:** Setting up the full suite (Admissions + Enrollment + Billing) can take several months.

## 6. Competitive Intelligence
- **SchoolAdmin vs. Blackbaud Enrollment:** SchoolAdmin is often praised for a better user interface (UI) and parent experience.
- **Best-in-Class Support:** Known for high-touch customer success and technical training.
- **The Winner:** Best for private/independent schools that need a professional, automated admissions funnel to drive enrollment numbers.
