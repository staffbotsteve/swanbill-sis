# Schoology Feature Spec

## Overview
Schoology is a comprehensive K-12 Learning Management System (LMS) owned by PowerSchool. It is designed to blend instructional tools, community engagement, and administrative oversight into a single platform.

## Core Features
- **Course Management:** Centralized hub for creating, organizing, and distributing course materials (assignments, quizzes, files).
- **Gradebook:** Supports traditional and standards-based grading. Includes automated grading for objective questions and grade passback to Student Information Systems (SIS).
- **Assignments:** Multimedia submission support (text, audio, video). Native integration with Google Drive and Microsoft OneDrive for "make a copy" workflows.
- **Assessments:** Advanced assessment engine with 20+ question types (highlight hotspot, ordering, etc.). Features built-in accessibility tools like calculators and text-to-speech.
- **Parent Portal:** High-visibility interface for parents to view grades, upcoming work, and teacher announcements.
- **Analytics:** Usage and performance reporting at the student, course, and school level.
- **Content Library:** Global and personal resource repositories for sharing and rating educational materials.
- **Mobile App:** Full-featured native apps for iOS, Android, and Kindle, supporting offline access and mobile-first submissions.

## Integration Capabilities
- **SIS Integration:** Deep, native integration with PowerSchool SIS; supports OneRoster for others (Aeries, Skyward, etc.).
- **LTI Support:** Fully compliant with LTI 1.1 and 1.3 standards.
- **Cloud Tools:** Seamless "App" integrations for Google Workspace, O365, YouTube, and Zoom.

## Pricing
- **Basic:** Free version available for individual teachers (limited features).
- **Enterprise:** Per-student annual fee, typically ~$10/student/year, plus implementation and training costs.

## Pain Points
- **UI/UX:** Often described as "clunky" or "Facebook-like" but dated. Navigation requires many clicks.
- **Reliability:** Performance slowdowns reported during high-concurrency periods (e.g., district-wide testing).
- **Integration Friction:** LTI implementations can be inconsistent; some users find SIS syncing "finicky."
- **Messaging:** Built-in messaging is basic and lacks advanced notification triggers.

## Competitive Intelligence
- **Vs. Canvas:** Schoology is more K-12 focused with better parent tools; Canvas is more modular and preferred in Higher Ed.
- **Vs. Google Classroom:** Schoology is a "full LMS" with administrative oversight; Google Classroom is a "workflow tool" for assignment collection.
- **Strategic Note:** Integrating with Schoology is essential for PowerSchool districts, but replacing it requires a significant "parent engagement" strategy to match its visibility tools.
