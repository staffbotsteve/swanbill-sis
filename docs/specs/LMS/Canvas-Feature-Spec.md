# Canvas LMS Feature Spec

## Overview
Canvas by Instructure is an extensible, cloud-native LMS known for its open-source core, modular design, and industry-leading adoption in Higher Ed and large K-12 districts.

## Core Features
- **SpeedGrader:** Highly efficient grading interface supporting inline annotations, rubrics, and video feedback.
- **Modules:** Allows for structured learning paths with prerequisites and requirements (e.g., "Must score 80% on Quiz A to unlock Lesson B").
- **Quizzes (New Quizzes):** Modern assessment engine with diverse question types, item banks, and deep analytics.
- **Outcomes & Mastery:** Tools to align assignments with specific standards/outcomes and track student progress toward mastery.
- **Learning Analytics:** Comprehensive dashboards (New Analytics) showing student participation and performance trends.
- **Open-Source Version:** The core LMS engine is open-source (AGPLv3), allowing for self-hosting and deep customization.
- **Mobile Suite:** Separate, high-quality apps for Students, Teachers, and Parents.

## Integration Capabilities
- **LTI Advantage:** Gold-standard compliance with LTI 1.3, making it the preferred platform for major publishers.
- **Canvas API:** Extremely well-documented REST API used for nearly all platform actions.
- **App Center:** "One-click" installation for hundreds of 3rd party integrations.

## Pricing
- **Free-for-Teacher:** Perpetual free version for individual educators.
- **Enterprise:** Custom quoted. Typically ~$5-$15 per user/year based on volume, plus significant implementation and "Success" fees.

## Pain Points
- **Learning Curve:** The "blank slate" nature of Canvas can be overwhelming for non-technical teachers.
- **UI Bloat:** Some users find the interface "sloppy" or requires too many clicks to perform simple tasks.
- **Administrative Overhead:** Usually requires a dedicated admin to manage permissions, Blueprints, and SIS syncing.
- **Cost:** Often the most expensive option once implementation and "add-on" modules (Studio, Catalog) are factored in.

## Competitive Intelligence
- **Vs. Schoology:** Canvas is "built to be customized"; Schoology is "built for K-12." Canvas is winning the "Scale" war in large US districts.
- **Vs. Google Classroom:** Canvas is the "Real LMS" schools move to when they outgrow Google Classroom's simplicity.
- **Strategic Note:** Integration with Canvas via LTI 1.3 is mandatory for credibility in the US K-12 market.
